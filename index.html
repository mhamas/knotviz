Key capabilities

GPU-accelerated

Smooth 60fps rendering powered by WebGL. Pan, zoom, and simulate force-directed layouts on graphs that would freeze other tools.

Property filtering

Filter nodes by any property — numbers, strings, dates, booleans, and tags. Combine multiple filters with AND logic.

Color gradients

Color nodes by property values with customizable palettes. Visualize patterns at a glance with continuous or discrete scales.

Edge analysis

Control edge visibility with percentage, max outgoing, and max incoming filters. Focus on the connections that matter.

How it works

1

Drop

Drag a JSON file onto the canvas. Your data stays in your browser — nothing is uploaded.

2

Explore

Pan, zoom, and run force-directed simulation to untangle the graph. Hover nodes to see details.

3

Analyze

Filter by properties, color by values, control edge visibility. Export the enriched graph when done.

Simple JSON input

Knotviz accepts a JSON file with nodes and edges arrays. Add optional properties for filtering and coloring.

transactions.json 
{
  "version": "1",
  "nodes": [
    { "id": "acme",  "label": "Acme Corp",  "properties": { "type": "company", "risk": 82 } },
    { "id": "sh-42", "label": "Shell #42",  "properties": { "type": "shell",   "risk": 95 } },
    { "id": "jdoe",  "label": "John Doe",   "properties": { "type": "person",  "risk": 40 } }
  ],
  "edges": [
    { "source": "acme", "target": "sh-42", "weight": 2.4 },
    { "source": "jdoe", "target": "acme",  "weight": 0.1 }
  ]
}

Full schema documentation is available in the app's schema dialog.
